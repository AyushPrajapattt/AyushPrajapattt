# 🚀 About Codesuu

💡 **Codesuu** is a platform designed to solve real problems in coding education:

🔥 **Problem 1:**  
Students struggle to participate in hackathons or coding rounds due to lack of proper platforms or guidance  

🔥 **Problem 2:**  
Students with medical or personal issues miss practical exams  

✅ **Solution:**  
Codesuu allows:  
✔️ Conducting **remote practical exams**  
✔️ Hosting **live coding sessions**  
✔️ Real-time **collaboration & evaluation**  
✔️ Easy **session creation & joining**
